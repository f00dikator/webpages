better breakdown of proto
0) Command Type – The integer enum mapped to specific function handlers inside the phMonitor service
1) Payload Length – The length of the payload in the message
2) Send ID – An arbitrary integer value passed in the message
3) Sequence ID – The sequence number of this message
4) Payload – The specific data the function handler within phMonitor will operate on
